# RADIUS 기반 중앙 인증 시스템 구축

## 1. 초기 구축

FreeRADIUS를 설치하고 네트워크 장비와 연동하여  
인증(Authentication) 및 권한(Authorization) 테스트를 진행하였다.

초기 테스트 결과 모든 장비에서 정상적으로 동작하였다.

---

## 2. 보안 이슈 확인

RADIUS 설정 과정에서  
사용자 인증 정보가 설정 파일에 평문으로 저장되는 이슈를 확인하였다.

- 계정 정보 노출 위험
- 보안 정책에 부적합

---

## 3. 개선 작업 (DB 연동)

보안 강화를 위해 인증 구조를 DB 기반으로 변경하였다.

- 사용자 계정 DB 저장
- 네트워크 장비(Client) 정보 DB 등록
- DB 접근 권한 제한 (관리자만 접근)

---

## 4. 문제 발생

DB 연동 후 테스트 진행 결과

- 일부 장비: 정상 동작
- L2 스위치: 로그인은 가능하지만 권한 적용 실패

즉, 인증은 성공하지만 Authorization이 정상적으로 동작하지 않는 문제 발생

---

## 5. 해결 과정

문제 해결을 위해 VTY 라인에 권한을 강제로 적용

```bash
line vty 0 4
 privilege level 15
```
---

## 6. 결과
RADIUS 기반 중앙 인증 시스템 구축
DB 기반 계정 관리 구조 적용
장비별 동작 차이를 고려한 문제 해결 경험 확보
